# React Springboot Integration

A full-stack application integrating a React frontend with a Spring Boot backend.

## Prerequisites

- Java 17
- Maven

## Technologies Used

- Spring Boot 3.5.13
- Spring Data JPA
- Spring Security
- Spring Validation
- Spring Web
- H2 Database (in-memory)
- Lombok

## Setup and Installation

1. Ensure you have Java 17 and Maven installed.
2. Clone or download the project.
3. Navigate to the project root directory.
4. Run the following command to build the project:

   ```
   mvn clean install
   ```

5. To run the application:

   ```
   mvn spring-boot:run
   ```

The Spring Boot application will start on port 8080 by default.

## Usage

Once the application is running, access it at: http://localhost:8080

Note: The React frontend should be built and placed in `src/main/resources/static/` for the integration to work properly.
